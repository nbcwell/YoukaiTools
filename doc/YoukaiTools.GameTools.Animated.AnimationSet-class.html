addAnimation(self, name, framedata, timedata=1, loops=-1, next_animation=None)

source code 

Adds a new animation.

Parameters:
  • name - The name of the animation.
  • framedata - A list where each element is a frame of animation. A frame can be any kind of data.
  • timedata - The length of each frame. If a list, each element is the amount of time spent on each frame. If a single number, then each frame will be set to that length.
  • loops - The number of times to loop the animation. If less than 1, the animation will be repeated infinitely.
  • next_animation - The name of the animation to be played after all of the loops. If None, it will stop.
