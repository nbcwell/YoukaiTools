Module LatLong

source code


Utilities for dealing with latitude and longitude values.
This module deals with 3 core types of objects:
1. Position is a basic lat or lon position. It is a tuple of length 4, with (degrees, minutes, seconds, direction).
   Direction is given as a string of a single letter "N", "S", "E", or "W".
2. Global Position is a numerically corrected position that has no direction string, It is a tuple
   of length 3, with (degrees, minutes, seconds). The function globalLatLon can produce these.
3. Radial Position is a given position in polar coordinates. It is simply a float. toRadians can produce these.

dist(latlon1, latlon2, radius)

source code 

Returns the distance between 2 latitude and longitudes on a sphere with a given radius. Latitude and longitude pairs should be given in tuples like: ((latdegree, latminute, latsecond), (londegree, lonminute, lonsecond))

Parameters:
  • latlon1 (Iterable) - The first latitude longitude pair.
  • latlon2 (Iterable) - The second latitude longitude pair.
  • radius - The radius of the sphere. The earth is 6378.1 km, or 3963.14 miles.
Returns: float

toGlobal(pos)

source code 

Takes a position (degrees, minutes, seconds) or (degrees, minutes, seconds, direction) Direction is a string with a letter "E", "W", "N" or "S". This function then converts it to a single global position without the direction.
